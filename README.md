# Small Enterprise Infrastructure Lab

Enterprise-style Windows environment built in VirtualBox.

## Objectives

- Deploy Active Directory
- Configure DNS + DHCP
- Enable NAT internet access
- Join client devices to domain
- Automate administration tasks

## Environment

Server:
- Windows Server 2022

Client:
- Windows 11 Enterprise

Services:
- AD DS
- DNS
- DHCP
- RRAS
