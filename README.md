# simple_ad
LAB AD Environment for Small to Medium Sized Enterprise


### Lab Environment

- Thinkcentre I5
- Windows 11 Pro
- Visual C++ 2019
- Virtual Box

## Virtual Environment
- Windows Server 2022 > Will serve as domain controller
- Windows 11 Enterprise > Will serve as computer which is part of the AD tree.
- Virtual Box HyperVisor


## Objectives
- Deploy Windows server environment with:
  - An active directory
  - Implement a DHCP service
  - Implement a DNS service
  - Configure a NAT service through the DNS to allow for outside web traffic
  - Script the creationg of a list of initial users
  - Automate onboarding and offboaring procedures
  - Apply basic security protocols

## Technologies

- Active Directory
- Group policy
- DNS
- NAT
- DHCP
- Powershell
